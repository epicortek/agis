# IETF Internet-Draft Material

This directory contains Internet-Draft working material for AgIS.

The internal AgIS protocol specification material may live under `/spec/`.

This directory is reserved for IETF-facing draft artifacts, including:

- Markdown draft sources
- generated RFCXML files
- generated text renderings
- idnits or author-tools output

Current draft:

- `draft-ayoub-agis-agent-identity-system-00.md`

The current draft is an individual Internet-Draft working file. It is not an RFC and does not imply IETF standardization status.
